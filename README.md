# Portfolio Repository

This repository contains the source code for the portfolio project.

## How to run the project locally

1. Open your terminal and navigate to the `yusuf` directory:
   ```bash
   cd yusuf
   ```

2. Install the necessary dependencies (if you haven't already):
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

This will run the application locally on `http://localhost:5173`.
